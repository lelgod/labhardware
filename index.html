Como Funciona

Este simulador mostra como um semáforo pode ser controlado por um programa em Assembly. Você pode escolher entre dois modos de execução:

Modo Passo a Passo: Executa o código linha por linha, mostrando cada instrução sendo processada.

Modo Automático: Executa o semáforo respeitando os tempos configurados para cada luz, sem mostrar as linhas de código individualmente.

Edite os tempos de duração de cada luz diretamente no código Assembly clicando nos valores destacados.
